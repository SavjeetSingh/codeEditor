# Python Code Editor 🧠💻

A multi-tab Python code editor built using Tkinter.

### ✨ Features

- 📝 Syntax highlighting
- 📑 Multiple tabs
- 🎨 Light/Dark theme toggle
- ⚙️ Code execution
- ✍️ Autocompletion with Jedi
- 🧹 Auto code formatting (Black)
